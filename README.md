# 🍹 Juice Shop Website

## 📌 Project Overview
This is a responsive Juice Shop website developed as a frontend project using HTML, CSS, and JavaScript. The website provides an attractive interface for customers to explore the menu, learn about the shop, and place orders.

## ✨ Features
- Responsive design
- Home page
- About page
- Menu page
- Contact page
- Order page
- Simple navigation bar
- Attractive images and layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📁 Project Structure
```
frontend-project/
│── index.html
│── about.html
│── menu.html
│── contact.html
│── order.html
│── style.css
│── script.js
│── images/
```

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
